# Chip-Category-Customer-Store-Trial-Analysis

##  Objective

This report analyzes chip purchasing behavior using transaction-level data to identify key customer
segments, product preferences, and store performance trends. The goal is to support data-driven
decision making for category growth

##  Tools Used

* Python (Pandas, Matplotlib)
* Power BI

##  Key Insights

1. Customer Insights
The analysis shows that 'OLDER SINGLES/COUPLES' segment contributes the highest revenue.
Different life stages exhibit varying purchasing patterns, with mainstream customers showing strong
engagement and consistent buying behavior.

2. Product Insights
The top-performing brand is 'Kettle'. Pack size and brand significantly influence purchasing
decisions, indicating opportunities for optimized product assortment and promotions.

3. Store Trial Analysis
Trial stores (77, 86, 88) were evaluated against control stores using statistical techniques such as
correlation and hypothesis testing. The comparison focused on total sales, number of customers,
and transactions per customer to identify performance uplift
